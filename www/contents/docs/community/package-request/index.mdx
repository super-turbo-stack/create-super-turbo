# Package Request

If you have a package that you think would be useful to add to the Create-Super-Turbo ecosystem, please open an issue on the [Create-Super-Turbo GitHub repository](https://github.com/super-turbo-stack/create-super-turbo/issues/new).

Please provide as much detail as possible about the package, including:

- The name of the package
- A link to the package's GitHub repository
- A description of the package's functionality
- Any additional information that may be helpful
